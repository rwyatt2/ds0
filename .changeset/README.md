# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the changesets repository](https://github.com/changesets/changesets).

## How to create a changeset

```bash
pnpm changeset
```

This will prompt you to select which packages have changed and what type of change it is:
- `patch`: bug fixes, docs fixes, small improvements
- `minor`: new components, new variants, new CLI commands
- `major`: breaking API changes

The changeset will be committed and included in the next release PR.
