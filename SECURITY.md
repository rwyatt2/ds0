# Security Policy

## Supported Versions

We support the latest minor version series.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Email: security@ds0.systems (or open a private GitHub Security Advisory)

Include:
* Steps to reproduce
* Impact assessment
* Affected package(s) and versions
* Proposed fix (if you have one)

We will acknowledge within 72 hours.
