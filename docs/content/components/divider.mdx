---
title: Divider
description: Visual separator between content.
---

## Usage
\`\`\`tsx
<Divider />
<Divider orientation="vertical" />
<Divider decorative={false} />
\`\`\`

## API
| Prop | Type | Default |
|---|---|---|
| orientation | horizontal \| vertical | horizontal |
| decorative | boolean | true |
