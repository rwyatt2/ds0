## Overview

Banner is a prominent, full-width notification bar that sits at the top of the page to communicate system-wide messages. It supports multiple severity levels, dismissibility, and optional action buttons. Use it for maintenance notices, feature announcements, promotional messages, or critical alerts.

## Usage

<ComponentPreview>
  <Banner variant="info">We're launching a new feature next week!</Banner>
</ComponentPreview>

```tsx
import { Banner } from '@ds0/react/banner';

export function Example() {
  return <Banner variant="info">We're launching a new feature next week!</Banner>;
}
```

## Accessibility

### Keyboard Interactions

| Key | Action |
|---|---|
| `Tab` | Move focus to action or close button |
| `Enter` / `Space` | Activate focused action or close button |
| `Escape` | Dismiss banner (if dismissible) |

### Screen Reader

Announces banner content. Critical variants (error, warning) use `role="alert"` for immediate announcement.

### ARIA

- `role="banner"` for informational banners
- `role="alert"` for error/warning banners
- Dismiss button has `aria-label="Dismiss banner"`

## AI Decision Guide

**Use Banner when:**
- Displaying system-wide maintenance notices or outage warnings
- Announcing new features, product updates, or promotions
- Showing cookie consent or legal compliance messages

**Don't use Banner when:**
- You need contextual inline feedback ã use Alert instead
