---
title: Badge
description: Small visual indicator for status, counts, or categories.
---

## Usage
\`\`\`tsx
<Badge>Default</Badge>
<Badge variant="success">Active</Badge>
<Badge variant="destructive">Error</Badge>
\`\`\`

## API Reference
| Prop | Type | Default |
|---|---|---|
| variant | default \| secondary \| destructive \| success \| warning \| outline | default |
| size | sm \| md | md |
