---
title: Popover
description: A non-modal overlay anchored to a trigger.
---

## Usage

```tsx
import { Popover } from '@ds0/react/popover';

<Popover>
  <Popover.Trigger>Open</Popover.Trigger>
  <Popover.Content>Content here</Popover.Content>
</Popover>
```

## Accessibility

| Key | Action |
|---|---|
| `Escape` | Closes the popover |
