## Overview
PricingTable displays a side-by-side comparison of pricing tiers with features, prices, and call-to-action buttons.

## Usage
```tsx
import { PricingTable } from '@ds0/react/pricing-table';
<PricingTable tiers={[{ id: 'pro', name: 'Pro', price: 29, period: 'mo', features: ['Unlimited'], highlighted: true }]} />
```
