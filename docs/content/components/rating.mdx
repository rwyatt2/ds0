## Overview

Rating provides an intuitive star-based input for collecting user feedback scores. It supports interactive selection with hover preview, read-only display mode, keyboard navigation, and customizable star count. Common uses include product reviews, satisfaction surveys, and content quality feedback.

## Usage

<ComponentPreview>
  <Rating value={3} />
</ComponentPreview>

```tsx
import { Rating } from '@ds0/react/rating';

export function Example() {
  const [value, setValue] = useState(0);
  return <Rating value={value} onChange={setValue} />;
}
```

## Accessibility

### Keyboard Interactions

| Key | Action |
|---|---|
| `Tab` | Focus the rating group |
| `ArrowRight` / `ArrowUp` | Increase rating by 1 |
| `ArrowLeft` / `ArrowDown` | Decrease rating by 1 |
| `Home` | Set to minimum (1) |
| `End` | Set to maximum |

### Screen Reader

Announces "Rating: X out of Y stars" when focused. Each star announces its position and checked state.

### ARIA

- `role="radiogroup"` on the container
- `role="radio"` on each star with `aria-checked`
- `aria-label` with descriptive text

## API Reference

### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `number` | `0` | Current rating value |
| `maxValue` | `number` | `5` | Maximum rating value |
| `onChange` | `(value: number) => void` | — | Called when rating changes |
| `isDisabled` | `boolean` | `false` | Whether the rating is disabled |
| `isReadonly` | `boolean` | `false` | Whether the rating is read-only |
| `allowHalf` | `boolean` | `false` | Whether half-star values are allowed |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size of the star icons |

## AI Decision Guide

**Use Rating when:**
- Collecting user feedback on products, services, or content
- Displaying average ratings in a read-only format
- Providing a satisfaction score input in surveys or forms

**Don't use Rating when:**
- You need binary thumbs up/down feedback → use FeedbackButtons instead
- You need a continuous numeric range → use Slider instead

## Related Components

- **Slider** — Use for continuous numeric ranges
- **Toggle** — Use for binary on/off feedback
- **Badge** — Use to display a static rating number as text
