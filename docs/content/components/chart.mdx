## Overview
Chart renders bar, line, and area charts using pure SVG with grid lines, axis labels, legend, and responsive sizing.

## Usage
```tsx
import { Chart } from '@ds0/react/chart';
<Chart type="bar" labels={['Jan','Feb','Mar']} datasets={[{ label: 'Revenue', data: [100, 200, 150] }]} title="Monthly Revenue" />
```

## API Reference
| Prop | Type | Default | Description |
|---|---|---|---|
| `type` | `'line' \| 'bar' \| 'area'` | `'bar'` | Chart type |
| `labels` | `string[]` | — | X-axis labels |
| `datasets` | `ChartDataset[]` | — | Data series |
| `title` | `string` | — | Chart title |
| `showLegend` | `boolean` | `true` | Show legend |
| `showGrid` | `boolean` | `true` | Show grid lines |
