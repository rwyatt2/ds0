## Overview

DataTable extends the basic Table component with interactive features like sorting, filtering, and row selection. It is designed for admin dashboards, data management interfaces, and any scenario where users need to interact with structured datasets.

## Usage

<ComponentPreview>
  <DataTable
    data={[
      { name: 'Alice', email: 'alice@example.com', role: 'Admin' },
      { name: 'Bob', email: 'bob@example.com', role: 'User' },
    ]}
    columns={[
      { accessorKey: 'name', header: 'Name' },
      { accessorKey: 'email', header: 'Email' },
      { accessorKey: 'role', header: 'Role' },
    ]}
  />
</ComponentPreview>

```tsx
import { DataTable } from '@ds0/react/data-table';

export function Example() {
  return (
    <DataTable
      data={users}
      columns={[
        { accessorKey: 'name', header: 'Name' },
        { accessorKey: 'email', header: 'Email' },
        { accessorKey: 'role', header: 'Role' },
      ]}
    />
  );
}
```

## Variants

### Default
Standard table with clean borders.

### Striped
Alternating row backgrounds for improved scanability in large datasets.

```tsx
<DataTable data={data} columns={columns} variant="striped" />
```

## Sizes

### Small
Dense layout for admin panels and compact UIs.

### Medium (default)
Standard spacing for most data display contexts.

### Large
Spacious layout for key data displays.

## States

### Loading
Shows a loading indicator while data is being fetched.

### Empty
Displays a customizable empty state message.

### With Selection
Enables row selection via checkboxes.

## Composition

### With Pagination
```tsx
import { DataTable } from '@ds0/react/data-table';
import { Pagination } from '@ds0/react/pagination';

export function PaginatedTable() {
  return (
    <div>
      <DataTable data={pageData} columns={columns} />
      <Pagination totalPages={10} currentPage={1} onPageChange={setPage} />
    </div>
  );
}
```

## Accessibility

### Keyboard Interactions

| Key | Action |
|---|---|
| `Tab` | Navigate between interactive elements |
| `Arrow Up/Down` | Navigate between rows |
| `Space` | Toggle row selection |
| `Enter` | Activate sort on column header |

### Screen Reader

- Table announces total row/column count
- Sort direction announced on column headers
- Selection state announced per row

### ARIA

- `role="grid"` on the table
- `aria-sort` on sortable column headers
- `aria-selected` on selectable rows
