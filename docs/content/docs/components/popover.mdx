## Usage

```tsx

<Popover>
  <Popover.Trigger>Open</Popover.Trigger>
  <Popover.Content>Content here</Popover.Content>
</Popover>
```

## Accessibility

| Key | Action |
|---|---|
| `Escape` | Closes the popover |
