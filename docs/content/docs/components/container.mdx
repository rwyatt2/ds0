  <ContainerPreview />

## Usage
```tsx
<Container><PageContent /></Container>
<Container size="sm"><NarrowContent /></Container>
```

## API
| Prop | Type | Default |
|---|---|---|
| size | sm \| md \| lg \| xl \| full | lg |
| padding | boolean | true |
| center | boolean | true |
