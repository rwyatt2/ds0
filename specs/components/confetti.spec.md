# Component Spec: Confetti
## 1. Overview
**Name:** Confetti  **Category:** Feedback
**Description:** A celebratory confetti animation triggered on user achievements, milestone completions, or special events.
## 2. Use Cases
### Use When
* Celebrating form submission or checkout completion
* Milestone achievements (100th customer, level up)
* Special events (birthdays, holidays, promotions)
### Don't Use When
* Regular UI feedback → use Toast or Notification
* You need to convey error states
* Users have reduced-motion preferences (auto-detected)
