# Component Spec: Badge

## 1. Overview

**Name:** Badge
**Category:** Data Display
**Description:** Small visual indicator for status, counts, or categories.

## 2. Use Cases

### Use When
* Showing status
* Displaying counts
* Categorizing content

### Don't Use When
* Interactive selection → use ToggleGroup
