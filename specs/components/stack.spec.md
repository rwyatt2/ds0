# Component Spec: Stack

## 1. Overview

**Name:** Stack
**Category:** Layout
**Description:** Arranges children in a vertical or horizontal stack with consistent spacing.

## 2. Use Cases

### Use When
* Vertical lists
* Horizontal rows
* Uniform spacing

### Don't Use When
* 2D grid layout → use Grid

## 3. Variants

> No named variants.
