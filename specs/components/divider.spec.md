# Component Spec: Divider

## 1. Overview

**Name:** Divider
**Category:** Layout
**Description:** Visual separator between content sections.

## 2. Use Cases

### Use When
* Separating content groups
* Visual breaks

### Don't Use When
* Spacing alone is sufficient → use Stack gap

## 3. Variants

> No named variants.
