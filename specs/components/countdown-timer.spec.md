# Component Spec: CountdownTimer
## 1. Overview
**Name:** CountdownTimer  **Category:** Data Display
**Description:** A countdown timer that displays remaining time in days, hours, minutes, and seconds with customizable formatting and completion callback.
## 2. Use Cases
### Use When
* Limited-time offers or flash sales
* Event start countdowns
* Session timeout warnings
### Don't Use When
* You need elapsed time → use a stopwatch utility
* You need a progress bar → use Progress instead
