# Phase 5: Recipes & Patterns

> **Instructions for AI:** Read ALL files in `.ai/` before starting.
> Recipes compose existing DS0 components into real-world UI patterns.
> They live in `recipes/` and are copy-paste via CLI: `npx ds0 add recipe auth-form`
> Each recipe follows a file structure similar to components but lighter — no primitives layer.
> Recipes MUST only use DS0 components. No raw HTML for anything DS0 covers.

See the full spec in the user request for Recipe 1–15 details.
